EXTENSIONES PARA FIREFOX

En la amplísima selección de extensiones de Firefox no podían faltar los complementos de accesibilidad. El proyecto Access Firefox agrupa todos los temas, extensiones y trucos que sirven para hacer de Firefox el navegador más accesible. Si pensabas que lo único que podías hacer era usar los atajos para aumentar el tamaño de letra (Control++ y Control+-), te equivocas.

Entre los complementos más destacados están HiViS Large, un tema de gran tamaño, Noise, que añade sonidos a todas las acciones, y el impresionante Firefox Accessibility Extension, un pack con más de cincuenta mejoras de accesibilidad. Pocos navegadores pueden presumir de mayor sensibilidad hacia los problemas de visión.

Los demás navegadores, por otro lado, parecen haberse quedado atrás. Para Firefox e Internet Explorer tienes BrowseAloud, un complemento gratuito que lee por ti el texto de las páginas web (incluyendo los ficheros DOC y PDF).
